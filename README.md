O projekcie

Strona wizytówkowa stworzona z myślą o firmie z branży integracji audio-wideo. Projekt obejmuje pełen cykl: od struktury i treści, przez wielojęzyczność, po wdrożenie produkcyjne.

Funkcje
🌍 Wersje językowe (PL/EN)
📱 W pełni responsywny design (mobile, tablet, desktop)
✉️ Formularz kontaktowy
⚡ Wdrożenie na Vercel z automatycznym CI/CD z GitHub
Stack technologiczny
Framework: Next.js / React
Język: TypeScript
Hosting: Vercel
Wersjonowanie: Git / GitHub
Budowa z wykorzystaniem: Claude Code
Status

🚧 Projekt w aktywnym rozwoju — kolejne zmiany i sekcje w toku.
